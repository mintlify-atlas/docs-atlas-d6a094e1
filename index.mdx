IEE Edu is the full-stack learning management system (LMS) powering the Instituto de Economía y Empresa. Built on Laravel 12, Inertia.js, and Vue 3, it delivers a complete educational experience — from a public course catalog and subscription plans to an interactive virtual classroom, final exams, PDF certificates, and a full admin dashboard for managing users, content, and payments.

<Steps>
  <Step title="Set up your local environment">
    Clone the repository, copy `.env.example` to `.env`, install PHP and Node dependencies, generate the app key, run migrations, and seed the database.
  </Step>
  <Step title="Create your first admin user">
    Use the built-in Artisan command `php artisan iie:make-admin your@email.com --name="Admin"` to create an admin account.
  </Step>
  <Step title="Configure your subscription plans">
    Set pricing via the `IIE_PLAN_TRIMESTRAL_PRICE`, `IIE_PLAN_SEMESTRAL_PRICE`, and `IIE_PLAN_ANUAL_PRICE` environment variables, then adjust them in the Admin → Settings panel.
  </Step>
  <Step title="Publish your first course">
    Log into the admin panel at `/admin/dashboard`, create a course with modules and lessons, and click **Publish** to make it available to students.
  </Step>
</Steps>
